What it is

The model is rented. The workspace doesn't have to be.

Most people's AI history is scattered across ChatGPT, Gemini, Claude, wherever the best model happens to live this month. AURI separates the experience from the provider, so the provider can change without the workspace, memory, or agents starting over.

01

One interface, any provider

Gemini today, Claude or OpenAI tomorrow, without losing history.

02

Data stays put

Conversations and files live in a database and object store you own, not a vendor's account.

03

Built for what's next

A foundation for personal agents, a knowledge base, and real memory, not just a chat window.

The stack

Five layers, each one replaceable on its own.

Presentation
what you see and type into
LobeChat
Intelligence
the model doing the thinking
Gemini 2.5 Pro / Flash
Memory
chats, agents, sessions
PostgreSQL
Knowledge
files and future knowledge base
RustFS
Source control
versioned, backed up, portable
GitHub

Each layer is independently swappable. Change the model provider and the memory layer doesn't notice. Change the storage backend and the chat history doesn't move. That separation is the whole point of AURI — it's an architecture decision, not a feature.
